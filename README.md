# Veloura Beauty Website

## Overview
This project is a responsive static website built using HTML and CSS.  
It represents a cosmetic brand showcasing products, services, and contact information.

## Features
- Clean and responsive design
- Navigation bar with multiple sections
- Product showcase with images and pricing
- Contact form
- Organized file structure using local assets

## Technologies Used
- HTML
- CSS

## Live Website
https://arena672.github.io/myProject/

## Purpose
This project was created to practice frontend development and understand website structure, layout, and design principles.
